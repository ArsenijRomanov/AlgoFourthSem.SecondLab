# TSP.Avalonia

GUI-проект на **Avalonia** для визуализации задачи о коммивояжёре поверх существующего ядра `TSP.Core`.

## Что внутри

- отдельный GUI-проект, не требующий изменений в `TSP.Core`
- стиль и структура, близкие к первой лабораторной
- поддержка двух семейств алгоритмов:
  - `Simulated Annealing`
  - `Ant Colony`
- визуализация графа, текущего и лучшего маршрута
- визуализация колонии и феромонов для ACO
- встроенная загрузка примеров `small6`, `berlin52`, `world666`
- загрузка пользовательских `.stp` файлов формата `33d32945 STP File, STP Format Version 1.00`
- экспорт лучшего маршрута
- экспорт изображения графа в PNG
- отдельное окно с графиком лучшего значения по итерациям

## Как подключить

1. Положи папку `TSP.Avalonia` рядом с папкой `TSP.Core`.
2. Добавь проект `TSP.Avalonia.csproj` в существующее решение `TSP.sln`.
3. Сделай `TSP.Avalonia` стартовым проектом.
4. Запусти приложение.

Структура каталогов должна получиться такой:

```text
AlgoFourthSem.SecondLab/
├─ TSP.Core/
└─ TSP.Avalonia/
```

## Основные возможности интерфейса

### Слева
- выбор встроенного графа или загрузка `.stp`
- конфигурация алгоритма
- критерии остановки
- кнопки `Initialize`, `Step`, `Step xN`, `Run`, `Reset`
- блок текущего результата

### Справа
- большая интерактивная область графа
- масштабирование колесом мыши
- перетаскивание полотна мышью
- `Fit`, `Peek best`, экспорт PNG, экспорт маршрута, окно графика истории

### Визуализация
- вершины показываются кругами с номерами
- веса рёбер и подписи вершин автоматически появляются только при достаточном увеличении и при включённых переключателях
- для SA:
  - текущий путь
  - лучший путь
  - отсутствующие рёбра подсвечиваются красным
- для ACO:
  - режим текущих путей колонии
  - режим карты феромонов
  - выдвижная нижняя панель с маршрутами муравьёв последней итерации

## Замечания

- Встроенные примеры ищутся в `TSP.Core/Examples`.
- Если пользовательский `.stp` файл не содержит координат всех вершин, GUI автоматически строит круговую раскладку.
- Само ядро `TSP.Core` не изменяется.
